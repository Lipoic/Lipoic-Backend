# Lipoic-Backend
The backend repository for lipoic, built with node.js + typescript + express.js.

## Development
### Get Started
Install [Node.js](https://nodejs.org) and [Yarn](https://yarnpkg.com/getting-started/install)

Run the commands
```shell
yarn install
```
### Start Server (dev)
```shell
yarn dev
```
### Start Server (product)
```shell
yarn start
```
### Build
```shell
yarn build
```
### ESLint
```shell
yarn lint
```

## Config
Create the `.env` file.
```env
PORT=8080
```
